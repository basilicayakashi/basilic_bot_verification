\# Discord Moderation \& Community Management Bot



A powerful Discord bot focused on community security, member verification, moderation tools, reaction roles, welcome messages, and free game notifications.



\## Features



\### 🔐 Member Verification



Protect your community with a customizable verification system.



\* Custom verification questions

\* Text and image-based verification forms

\* Staff approval workflow

\* Private moderation channels

\* Manual member verification

\* Verification timeout management



\### 🛡️ Shared Blacklist System



Moderators can identify problematic users across multiple communities.



\* Global user lookup

\* Shared blacklist information

\* Blacklist reasons tracking

\* Cross-server moderation assistance



\### 🚨 Spam Detection



Automatically detect suspicious activity and alert moderators.



\* Configurable detection thresholds

\* Dedicated alert channels

\* Staff role notifications

\* Adjustable monitoring windows



\### 👋 Welcome Messages



Create personalized welcome messages for new members.



\* Use any existing Discord message as a template

\* Supports member mentions with `{{mention}}`

\* Preview configured welcome messages

\* Easily update or remove messages



\### 🎭 Reaction Roles



Allow members to self-assign roles.



\* Category-based role panels

\* Custom emojis support

\* Automatic panel updates

\* Easy role management



\### 🎮 Free Games Notifications



Automatically share free game promotions.



\* Steam support

\* Epic Games support

\* Automatic expiration cleanup

\* Dedicated publication channels



\### ⚙️ Multi-language Support



The bot supports multiple languages:



\* English

\* French

\* Spanish

\* German

\* Polish



\## Commands



The bot includes an integrated help system:



```text

/help

```



Available sections:



```text

about

verification

welcome-message

spam

reaction-roles

free-games

permissions

```



\## Built With



\* TypeScript

\* Discord.js

\* SQLite



\## Goal



This project was created to provide Discord communities with a complete moderation and onboarding solution while remaining simple to configure and maintain.
